# Arrata-Bot

A bot for managing Arrata games.

## Usage

### Dice Rolls

```console
!a -a [Advantage (Optional)] -d [Disadvantge (Optional)] r|roll [Quality][Quantity] [Ob (Optional)]
!a roll b10 ob5
!a roll -a 1 b10
!a r -d 10 A30
```

### Random Character Generation

```console
!a char [Stat Buy points (Optional)]
!a char 30

```
